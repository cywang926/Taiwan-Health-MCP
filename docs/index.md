# Taiwan Health MCP Server

台灣醫療健康資料整合 MCP 伺服器，提供 ICD-10、LOINC、SNOMED CT、臨床指引、健康食品、食品營養、TWCore 與 FHIR Condition 相關工具。

## 目前能力

- ICD-10 診斷與手術碼查詢
- LOINC 與檢驗數值判讀
- SNOMED CT 概念與映射
- 台灣健康食品與食品營養查詢
- 臨床指引搜尋與分段查詢
- TWCore CodeSystem 查詢
- FHIR Condition 產生與驗證
